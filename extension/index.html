This Firefox extension requires prior installation of the JAMScript speculative execution system to the browser (currently supporting Firefox 31.4esr). See the project at https://github.com/blackoutjack/jamscript for more information.

Given a prior installation of the JAMScript speculative execution framework, this extension allows the browser user to choose from several prefabricated policies to enforce. The policy enforcement starts with the next webpage that's loaded.
